# dotfiles
Personal configuration files for Linux.

## Packages
```txt
alacritty  Terminal
bash       Shell
dunst      Notifications
git        Version control
i3         Window manager
polybar    Status bar
redshift   Display color temperature adjustment
ssh        Remote shell
systemd    Event API
tmux       Terminal multiplexer
vim        Text editor
workrave   RSI prevention
xorg       GUI environment
```
